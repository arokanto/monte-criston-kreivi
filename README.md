# Monte-Criston kreivi

A fixed epub of the Finnish translation of Count of Monte-Cristo

If you only care about the epub, you can find the updated version at the root of the project.

## Development

This project uses Deno so make sure you have it installed, by for example running:

```bash
brew install deno
```

To build the epub version from source, run

```bash
deno run build:epub
```
